# SchoolAI Tutor

## Problem Statement

Students often struggle to receive personalised instruction and timely feedback in large classrooms. An intelligent tutoring system can adapt to each learner’s needs and provide interactive support.

## Tech Stack

- Python
- PyTorch
- Streamlit
- Transformers
- Matplotlib

## Installation

1. Clone this repository.
2. Create a virtual environment and activate it.
3. Install the dependencies: `pip install -r requirements.txt`.
4. Run the project: `python main.py`.

## Usage

Run the script with the default settings to see sample output. Modify the code to integrate with real data sources or user interfaces.
